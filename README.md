# trained-immunity.com

An interactive, beginner-friendly introduction to **trained immunity** — the discovery that the innate immune system can remember.

A single self-contained `index.html`: editorial scrollytelling, a live canvas immune-cell field (chemotaxis + phagocytosis), and interactive set-pieces (a "train the cell" mechanism walkthrough, animated evidence stats, and more).

## Story arc
Hero → 01 The Dogma → 02 The Accident → The Key Man (Mihai Netea) → 03 The Mechanism → 04 What Triggers It (BCG + β-glucan) → 05 The Proof → 06 The Big Picture.

## Stack
- Plain HTML/CSS/JS — no build step, no dependencies
- Type: IBM Plex Serif / Sans / Mono
- Hosting: Vercel (static) · DNS: Cloudflare

## Local
Open `index.html` in a browser. That's it.

## Sources
Netea, Quintin & van der Meer, *Cell Host & Microbe* (2011, 2017), plus Netea lab lectures & interviews.
